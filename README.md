#Инструкция по применению
---
#Установка

 - Скачать zip-архив
 - Разорхивировать
 - открыть папку 'dist'
 - Запустить 'DeepSeekChat.exe'

---
#Управление
 - Для вызова окна нажать - 'ctrl + alt + space'
 - Для закрытия - esc
---
Может появиться уведомление от антивируса виндоус, это нормально,
 т.к. данное приложение напрямую работает с клавиатурой (Для горячих клавишь),
 брендмауер может на него жаловаться, игнорируйте)
